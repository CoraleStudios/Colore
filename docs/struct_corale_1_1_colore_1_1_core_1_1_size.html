Detailed Description

Dynamic size type for interacting with Razer's native code. Emulates size_t from C.

This struct will use either ulong or uint as its base, depending on compile-time or run-time architecture. On 64-bit systems, if compiled in x64, the base will be ulong. On 32-bit systems, if compiled in x86, the base will be uint. On 64- and 32-bit systems, if compiled in Any CPU, the base will be uint, with context-sensitive code doing run-time conversions.
